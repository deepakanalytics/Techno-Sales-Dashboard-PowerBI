# Techno Sales Dashboard (Power BI)

## Project Overview

This Power BI dashboard provides an interactive analysis of technology product sales across India. The dashboard helps track sales performance, profit, quantity sold, brand contribution, and regional sales trends.

## Tools Used

- Power BI
- Excel
- DAX
- Data Modeling

## Key Metrics

| KPI | Value |
|------|------|
| Total Sales | 99M |
| Total Cost | 76M |
| Total Profit | 23M |
| Total Quantity | 13K |
| Total Transactions | 5.1K |

## Dashboard Features

- Category-wise Sales Analysis
- Brand Performance Analysis
- State-wise Sales Distribution
- Supervisor-wise Filtering
- Interactive KPI Monitoring

## Key Insights

- Monitor category generated the highest sales volume.
- Intel was the top-performing brand.
- Total profit reached 23M.
- Multiple Indian states contributed significantly to overall revenue.
